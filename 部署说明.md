# 阿宗国际佛学院 - 部署说明

## 快速部署（推荐）

### 第一步：在 GitHub 上创建空仓库

1. 登录 https://github.com
2. 点击右上角 "+" → "New repository"
3. Repository name 填写：`azong-buddhist-college`
4. 选择 "Public"（公开）
5. **不要**勾选 "Add a README file"
6. 点击 "Create repository"

### 第二步：运行部署脚本

在终端中执行以下命令：

```bash
# 进入项目目录
cd "/Users/cl/Documents/trae cn/trae-cn/trae/azong-buddhist-college"

# 运行部署脚本
./deploy.sh
```

脚本会提示您输入 GitHub 邮箱和用户名（首次运行），然后自动完成部署。

### 第三步：启用 GitHub Pages

1. 访问：https://github.com/garang1318/azong-buddhist-college
2. 点击顶部的 "Settings"（设置）
3. 左侧菜单点击 "Pages"
4. "Source" 选择 "Deploy from a branch"
5. "Branch" 选择 "main"，文件夹选择 "/ (root)"
6. 点击 "Save"
7. 等待 2-3 分钟

### 第四步：访问网站

- **前台页面**：https://garang1318.github.io/azong-buddhist-college/
- **后台管理**：https://garang1318.github.io/azong-buddhist-college/admin.html

## 手动部署（备选）

如果脚本运行失败，可以手动执行：

```bash
# 进入项目目录
cd "/Users/cl/Documents/trae cn/trae-cn/trae/azong-buddhist-college"

# 初始化 Git 仓库
git init

# 添加远程仓库
git remote add origin https://github.com/garang1318/azong-buddhist-college.git

# 添加文件
git add index.html admin.html config.json README.md

# 提交
git commit -m "Initial commit"

# 推送（需要输入 GitHub 用户名和密码）
git push -u origin main
```

## 常见问题

### 1. 提示 "Permission denied"

给脚本添加执行权限：
```bash
chmod +x deploy.sh
```

### 2. 推送时要求输入密码

GitHub 已不支持密码登录，需要使用 Token：
1. 访问 https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 勾选 "repo" 权限
4. 生成后复制 Token
5. 在终端输入 Token 代替密码

### 3. 仓库已存在

如果 GitHub 上已有同名仓库，可以先删除或更换仓库名。
